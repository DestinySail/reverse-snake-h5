# 逆光蠕行

一个基于 `Vite + TypeScript + Canvas` 的反向贪食蛇 H5 游戏。

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 测试

```bash
npm test
```

## GitHub Pages

仓库内已包含 `.github/workflows/deploy-pages.yml`。

- 推送到 `main` 分支后会自动构建并部署到 GitHub Pages
- 首次使用时，需要在 GitHub 仓库设置中将 `Pages` 的发布源切换为 `GitHub Actions`
